# Uitenhage Super Steel (USS) Website

Official website for Uitenhage Super Steel - ISO 9001:2015 certified structural steel fabricators established in 1972.

## 🏗️ Features

- **Industrial Dark Mode Design**: Modern, premium aesthetic with `bg-gray-900` sections
- **Interactive Carousels**: 
  - Services carousel with 4 core offerings
  - Projects carousel showcasing 6 major installations
- **Video Hero Section**: Optimized autoplay background video
- **Fully Responsive**: Mobile-first design with Tailwind CSS
- **Quality & Certifications Section**: Dedicated ISO 9001:2015 compliance showcase

## 🎨 Technology Stack

- **HTML5**: Semantic markup
- **Tailwind CSS**: Via CDN for rapid styling
- **Vanilla JavaScript**: Lightweight carousel logic
- **Font Awesome**: Icon library

## 🚀 Deployment

This is a static website that can be deployed to:
- **Vercel** (recommended)
- Netlify
- GitHub Pages
- Any static hosting provider

### Deploy to Vercel

1. Push code to GitHub
2. Import repository in Vercel
3. Deploy with default settings
